¿Qué es 2FA?

La Autenticación de Dos Factores (2FA) añade una capa extra de seguridad a tu cuenta. Además de tu contraseña, necesitarás un código de verificación generado por una aplicación como Google Authenticator para iniciar sesión.

Ventajas:

  • Mayor protección contra accesos no autorizados
  • Los códigos cambian cada 30 segundos
  • Funciona sin conexión a internet

Recomendaciones:

  • Guarda tu código de recuperación en un lugar seguro
  • Usa una aplicación de autenticación confiable
  • Nunca compartas tus códigos con nadie
